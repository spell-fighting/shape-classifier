# Shape Classifier

Uses [Google QuickDraw](https://github.com/1991viet/QuickDraw) dataset to recognize between :

- circle
- hourglass
- square
- star
- triangle

Used to use dataset generated by [Shape Drawer](https://shape-drawer.netlify.com/).

## Installation

+ `pipenv install`

## Usage

+ `cd src/`
+ `python download_data`
+ `python train.py`
